h2. Glass Sitecore Mapper Razor



Glass Sitecore Mapper Razor is an extension to Glass Sitecore Mapper for "Sitecore":http://www.sitecore.net that creates a simple and easy to use integration for Glass Sitecore Mapper, Razor view engine and Sitecore.

You can keep up to date by following "Glasslu":http://twitter.com/#!/Glasslu on twitter.
